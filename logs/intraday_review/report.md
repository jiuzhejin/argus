# Intraday signal review

- Coverage: `2026-05-14` to `2026-06-26`, `28` trading snapshots, `29` ETFs
- Buy event: first day entering `★ 买入信号` or `◆ 趋势持有`
- Sell event: explicit `🔴 止损`; if absent after a buy, first day falling into `✗ 趋势偏弱` is marked as proxy sell
- Evaluation rule:
  - Buy `有效`: 5-day return > 0; `存疑`: 5-day return <= 0 but max gain within 5 days > 0; `失效`: max gain within 5 days <= 0
  - Sell `有效`: 5-day return < 0; `存疑`: 5-day return >= 0 but min return within 5 days < 0; `失效`: min return within 5 days >= 0
